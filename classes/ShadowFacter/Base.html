fact(key, value=nil, confine_args={}, &block)

Define a fact in Facter using a value or block. Can be confined with a hash of Facter keys (namespace_key) and value.

Examples: 

  fact :tea, "oolong"
  fact :tea, "puerh", {:drinks_season => "winter"}
  fact(:rand) { rand }
