ShadowFacter

ShadowFacter is a Ruby DSL for Facter, extracted out of the work we at Rails Machine are doing on Moonshine.

ShadowFacter provides a DSL for creating facts and processing them using Facter. For documentation on writing these manifests, please see ShadowFacter::Base.

A binary is provided to parse facts.
